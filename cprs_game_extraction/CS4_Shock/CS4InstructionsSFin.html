In dem vorherigen Teil dieses Spiels bestand die Chance, dass Ihr Gemeinschaftswald von einem Klimaschock betroffen ist und dadurch Ihre Ressourcenkapazität verringert wird.

In diesem Teil ist allerdings ein Klimaschock direkt zu Beginn eingetreten, wodurch ein Waldbrand ausgelöst wurde. Das bedeutet, dass nun die ursprüngliche Waldgröße {{C.INITIAL_TREES_OLD}} um 50% reduziert wird, sodass nur {{C.INITIAL_TREES_OLD}}  × {{C.MULTIPLIER_SHOCK}} = {{C.INITIAL_TREES}} Bäume übrig bleiben. Ihre Gruppe startet das Spiel also nun nicht mit 100 Bäumen, sondern mit 50 Bäumen.

Ihre Entscheidungssituation ist wie in Teil 1, mit einer Ausnahme:

  • Die maximale Anzahl an Bäumen, die Sie in jeder Runde entnehmen können, ist kein fixer Betrag mehr, sondern hängt von dem Bestand des Waldes ab. Mit anderen Worten, je weniger Bäume im Wald vorhanden sind, desto geringer auch Ihre Möglichkeiten, Bäume zu entnehmen.

    Die minimale und maximale Anzahl an Bäumen, die Sie pro Runde in verschiedenen Szenarien entnehmen können, ist unten in der Tabelle aufgelistet:

    Aktuelle Anzahl an Bäumen pro Runde Minimal Maximal
    ≥21 0 7
    ≥18 0 6
    ≥15 0 5
    ≥12 0 6
    ≥9 0 3
    ≥6 0 2
    ≥3 0 1
    von 0 bis <3 0 0
  • Die Anzahl an Bäumen, die Sie in einer Runde fällen dürfen, wird Ihnen vor Ihrer Entscheidung mitgeteilt.
  • Falls der Gemeinschaftswald in einer beliebigen Runde unter 3 Bäume fällt, können Sie und Ihre Gruppenmitglieder die verbliebenen Bäume nicht entnehmen, es sei denn, der Wald wächst in den folgenden Runden auf mindestens 3 Bäume wieder an.

Ansonsten gibt es keine Änderungen im Vergleich zu Teil 1. Die Anzahl der Punkte, die Sie für das Abholzen von Bäumen und die verbleibenden Bäume im Wald erhalten, bleibt gleich. Ihre Gruppe bleibt dieselbe wie zuvor.
